Auctions

  • /auctions

    Retrieve all auctions

    Parameters

    example: /auctions?sort=-id,-duration&cat_id=1,2,3&page=1&page_count=2

    parameter type description
    sort

    string

    		 retrieve data in an ordered fashion. ie: "-id" to retrieve data ordered by id descending

    Possible sorts: id, cat_id, auc_art_id, start, end

    filter

    string

    		 retrieve a portion of the availlable data based on your filtered values.

    Possible filters: cat_id, status (0:active, 1:upcoming, -1:expired)

    page

    int

    		 retrieve specific page of data
    page_count

    int

    		 retrieve set amount of auctions per page

    Responses

    Description
